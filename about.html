About Chelsea FC

Chelsea Football Club is a professional football club based in Fulham, London. Founded in 1905, the club competes in the Premier League, the top division of English football.

Key facts:

  • Home Ground: Stamford Bridge (capacity: 40,343)
  • Major Honors: 6 Premier League titles, 8 FA Cups, 5 League Cups, 2 UEFA Champions Leagues
  • Nickname: The Blues
  • Owner: Todd Boehly, Clearlake Capital

Project Summary

This website was created as a final project for the HTML & CSS course at Jamburiya University. It demonstrates:

  • HTML5 semantic structure
  • CSS styling and layout techniques
  • Responsive design principles
  • Form validation and accessibility features

The project showcases all required elements including multiple pages, a gallery with images and data tables, a booking form, help documentation, and this about page.
